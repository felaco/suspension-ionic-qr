Rear Suspension Handling Tool Assembly

TIP

The rear suspension handling tool is designed to help with the installation of the rear suspension.

A fork lift must be used to lift andmove the suspension handling tool to position and align the upper eye bearing with the frame clevis.

 CAUTION

The rear suspension handling tool is not validated to be used for the removal of the rear suspension

Use the recommended tools shown for the rear uspension removal.

Step 1

As a safety measure, use a chain link (5) to attach the rear suspension handling tool (3) to the forklift mast

Step 2

Using the forklift, lift the rear suspension handling tool (3) above the ground and tilt the forks to prevent the tool from sliding from the forks.

Step 3

Safely assemble the rear suspension (1) to the handling tool (3), and use the security strap (4) to secure the rear suspension (1).

Step 4

Rotate the upper eye so the transducer cover is oriented toward the rear of the haul truck.

Step 5

Use the lever arm (2) to orient the upper eye perpendicular to the lower eye
